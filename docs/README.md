# 기능 명세서
- [x] : 카드 종류 하트, 클로버, 다이아, 스페이드다.
- [x] : 카드 숫자 2 ~ 10, A, J, Q, K 다.
- [ ] : 카드 A 는 1 혹은 11이다.
- [x] : 카드 J, Q, K 는 10이다.
- [x] : 카드의 경우의 수는 52개다.
- [ ] : 52장의 카드를 섞어서 플레이어들에게 2장씩 분배한다.
  - [ ] : 방식 1 : 중복 X
  - [ ] : 방식 2 : 중복 O
- [x] : Cards 에 card 를 넣는 기능
- [x] : Cards 에 다른 Cards 를 더하는 기능
- [x] : Cards 에 card 가 있는지 확인하는 기능
